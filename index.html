GearmanBundle

                            This documentation is currently in construction. Sorry for the inconvenience.

GearmanBundle is a bundle for symfony2 intended to provide an easy way to support developers who need use job queues. For example mail queues, Solr queues or Database upload queues.

All code is Symfony2 Code formatted, so every pull request must validate phpcs.sh script. Results are placed in Project/Reportings/codesniffer.log. You can follow this instructions to install all needed software and execute script from project root.

				sh ./phpcs.sh
